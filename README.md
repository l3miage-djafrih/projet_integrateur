#Génération de la matrice de distances :
    Le service generationMatrice.ts"définit la fonction generateMatriceFromAdresses, qui prend en entrée une liste d’adresses. La fonction commence par extraire les coordonnées [longitude, latitude] nécessaires à la génération de la matrice. Ensuite, elle effectue un appel à l’API Matrix d’OpenRouteService, qui se charge de générer la matrice sous format JSON. Ce JSON est ensuite transformé en objet TypeScript à l’aide de la bibliothèque Zod, exactement comme cela a été montré en cours.
#limitation :
    l'api ne peut pas prendre plus de 59 points d'adresses différents pour générer la matrice ,à ce stade on est obligé faire des manipulation sur le code pour casser le nombre d'adresses
